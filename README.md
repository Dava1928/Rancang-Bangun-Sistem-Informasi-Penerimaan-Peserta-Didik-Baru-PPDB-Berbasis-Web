# Sistem PPDB MTS SA BINARAHMAH

## Teknologi
- PHP Native
- MySQL
- Bootstrap 5
- JavaScript

## Fitur
- Login multi-user
- Pendaftaran online
- Upload berkas
- Verifikasi admin
- Cetak bukti pendaftaran

## Struktur Sistem
- admin = halaman admin
- auth = autentikasi
- config = koneksi database
- assets = css/js/upload
